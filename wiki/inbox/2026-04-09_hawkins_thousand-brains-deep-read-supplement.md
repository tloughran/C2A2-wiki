## Summary
This supplementary proposal extends the analysis of Clay, Leadholm, and Hawkins (2412.18354) beyond the two PRS triplets already proposed. A deeper structural reading — cross-referenced against all 11 C2A2 traditions — reveals six additional PRS-grade insights, including one paradigm-shift candidate for C2A2's own architecture and three previously unidentified cross-tradition bridges. The existing proposals correctly flag Friston, Levin, and Wolfram connections; this supplement identifies new bridges to Hoffman, Kastrup, Fredrickson, Stump, and — most significantly — to the C2A2 system itself.

## Why This Matters
The Thousand Brains paper is not just a contribution to Hawkins' tradition. Its architectural claims — semi-independent learning modules, a universal messaging protocol, reference frames as the organizing principle for all knowledge, and sensorimotor (not passive) learning — describe the very structure C2A2 is building. This paper should be read as a design document for the tradition-accelerator network, not merely as a neuroscience contribution.

## Candidate PRS Triplets

PRS-CANDIDATE-01:
  Problem: C2A2's 13-agent architecture was designed by intuition — 11 tradition agents independently maintaining models, communicating via dispatches, with integration layers (Master Agent, Pattern Detector) synthesizing across them. But there is no formal computational theory validating this design or predicting its failure modes.
  Resource: The Thousand Brains architecture is a formally described system with the same structure: semi-independent learning modules (= tradition agents), each building complete models of objects (= tradition wikis), communicating via a Cortical Messaging Protocol (= dispatch system), reaching consensus through voting (= Pattern Detector evaluation). The CMP specifies what modules must communicate: hypothesized identities and locations within reference frames — not raw observations.
  Solution: C2A2 IS a Thousand Brains system. This structural homology is not metaphorical — it is architecturally precise. The paper provides testable design principles: (a) dispatches should include each agent's current "location" in its tradition's reference frame, not just findings; (b) the Master Agent should implement explicit voting/consensus protocols rather than narrative integration; (c) lateral (non-hierarchical) communication between tradition agents may be as important as hierarchical routing through the Master Agent — this is the heterarchy insight applied to C2A2 itself.
  Confidence: High
  Evidence: Structural mapping: learning module = tradition agent; reference frame = PRS triplet structure; CMP = dispatch protocol; voting = Pattern Detector evaluation; sensorimotor loop = ingest → extract → dispatch → integrate cycle. The paper's Fig. 1 architecture diagram maps onto C2A2's agent diagram node-for-node.

PRS-CANDIDATE-02:
  Problem: Hawkins argues all knowledge is organized in reference frames — spatial, conceptual, or abstract. But the C2A2 system uses PRS triplets (Problem-Resource-Solution) as its universal knowledge format without theoretical justification for why this particular structure should work.
  Resource: Hawkins' claim that reference frames are the universal organizing principle for ALL knowledge in the neocortex — not just spatial knowledge. A reference frame provides a coordinate system within which features are located relative to each other; it is what makes knowledge navigable and transferable.
  Solution: PRS triplets ARE reference frames. A Problem defines the origin (where you are), a Resource defines the axes (what tools are available), and a Solution defines the target location (where you need to get). This means PRS is not an arbitrary format but reflects the deep structure of how intelligence organizes knowledge. It also suggests PRS should be extended: triplets should include "displacement vectors" — the path from Problem through Resource to Solution — not just three static points.
  Confidence: Medium
  Evidence: Hawkins 2024 §3 (reference frames as universal organizing principle); the structural parallel between spatial navigation (location → movement → destination) and knowledge navigation (problem → resource → solution) is the same displacement-encoding mechanism.

PRS-CANDIDATE-03:
  Problem: Hoffman's interface theory claims perception is a fitness-tuned user interface, not a window onto reality. Hawkins' learning modules build "CAD-like spatial models" through active exploration. Are these the same claim in different vocabularies?
  Resource: In the Thousand Brains paper, learning modules do NOT build photographs or veridical representations of reality. They build structured models — reference frames with features attached at specific locations — that are useful for the organism's interaction with the world. The models are evaluated by their predictive utility, not their correspondence to objective reality.
  Solution: Hawkins' reference frames ARE Hoffman's interfaces, implemented in cortical hardware. Both frameworks agree: (a) what the brain builds is not a picture of reality but a useful model; (b) the model is structured by the agent's needs, not by the world's intrinsic structure; (c) action (sensorimotor interaction) is essential to model construction. This is a previously unidentified cross-tradition bridge: Hoffman provides the evolutionary/philosophical argument for WHY reference frames are interfaces (fitness beats truth); Hawkins provides the neuroanatomical account of HOW the brain implements them (cortical columns as reference frame builders).
  Confidence: High
  Evidence: Hawkins' explicit description of learning modules as building "CAD-like" models, not photographs; Hoffman's fitness-beats-truth theorem (2015); both reject naive realism and both place action at the center of perception.

PRS-CANDIDATE-04:
  Problem: The Thousand Brains paper insists that sensorimotor interaction — active movement through the world — is essential for learning. Passive data ingestion cannot substitute. But C2A2's tradition agents currently operate in pure ingestion mode: they read documents, extract PRS triplets, and dispatch findings. They do not actively probe.
  Resource: Hawkins' sensorimotor learning principle: learning modules must generate predictions, execute actions that test those predictions, and update their models based on the result. Each cycle moves the agent to a new "location" in the reference frame, revealing features that were invisible from the previous location.
  Solution: C2A2 agents should be redesigned to include an active inquiry cycle: (a) after ingestion, each tradition agent generates hypotheses about what another tradition would say about the same material; (b) the Master Agent routes these hypotheses to the relevant tradition agent; (c) the receiving agent evaluates the hypothesis against its own reference frame and responds. This converts C2A2 from a passive recording system into a sensorimotor learning system — the agents "move" through the inter-tradition space by actively testing cross-tradition predictions, just as Monty's learning modules "move" their sensors to build 3D models.
  Confidence: Medium
  Evidence: Hawkins 2024 §2 (sensorimotor learning as fundamental); the observation that C2A2's current ingestion-only architecture violates Hawkins' central design principle.

PRS-CANDIDATE-05:
  Problem: Fredrickson's broaden-and-build theory shows that positive emotions widen the scope of attention and thought. But "broadened attention" has no mechanistic account at the neural level — it is described phenomenologically, not computationally.
  Resource: In the Thousand Brains framework, the number of simultaneously active learning modules determines the richness of the current world-model. If more cortical columns are actively contributing their reference frames, the organism has a more multi-faceted, integrated representation. Narrowed attention = fewer active columns; broadened attention = more active columns.
  Solution: Broaden-and-build IS reference frame multiplicity. Positive emotions, via neuromodulatory changes (dopamine, norepinephrine), may increase the number of cortical columns that remain in active hypothesis-generation mode rather than being suppressed by lateral inhibition. This provides the first mechanistic neural account of Fredrickson's core observation: broadened attention = more active reference frames = richer integration. It also predicts that positive states should improve cross-tradition understanding (more reference frames from different traditions can be held simultaneously) — directly relevant to C2A2's second-first-language question.
  Confidence: Speculative
  Evidence: Fredrickson 2001 (broaden-and-build); Hawkins 2024 (multiple active learning modules as basis of rich perception); known effects of dopaminergic signaling on cortical column activity. No direct empirical test yet — this is a prediction.

PRS-CANDIDATE-06:
  Problem: Kastrup's analytic idealism describes individual minds as "dissociated alters" of universal consciousness, with Markov blankets as the dissociative boundary (FINDING-009). But this model has been applied only at the scale of individual organisms — not within a single brain.
  Resource: The Thousand Brains paper describes ~150,000 cortical columns, each building its own COMPLETE model of the world — not a fragment, but a whole. Each column is, in a specific sense, its own "mind" — it has its own reference frame, its own sensory processing, its own predictions. The columns interact but are not merged.
  Solution: Cortical columns are alters within a single brain. The Thousand Brains architecture is Kastrup's DID model applied at the cortical level: each column is a "dissociated" world-model that interacts with other columns through a communication protocol (CMP) but maintains its own independent processing. Consciousness at the brain level is the INTEGRATION of these column-level alters via the voting/consensus mechanism — exactly as Kastrup's universal consciousness would integrate individual minds. This extends FINDING-009 (Kastrup × Friston Markov blanket) downward: the same dissociation-integration dynamic operates at EVERY scale, from universal consciousness to individual organisms to individual cortical columns.
  Confidence: Speculative
  Evidence: Hawkins' explicit claim that each cortical column builds a COMPLETE model (not a fragment); Kastrup's DID model; the structural parallel between inter-column voting and inter-mind integration.

## Cross-Tradition Signals

**Hoffman (NEW — not in existing proposals):** Hawkins' reference frames are Hoffman's interfaces implemented in cortical hardware. Both reject naive realism; both center action in perception; both describe models structured by utility, not correspondence. Flag for Hoffman Agent: this may be the strongest neuroscience-to-philosophy bridge in the network after Levin × Friston. See PRS-CANDIDATE-03.

**Kastrup (NEW — not in existing proposals):** Thousand Brains architecture instantiates Kastrup's DID model within a single brain. Extends FINDING-009 to cortical column scale. Flag for Kastrup Agent and Pattern Detector. See PRS-CANDIDATE-06.

**Fredrickson (NEW — not in existing proposals):** Broaden-and-build = reference frame multiplicity. First mechanistic neural account of Fredrickson's core observation. Flag for Fredrickson Agent. See PRS-CANDIDATE-05.

**Stump (implicit):** Thomistic "form" organizes matter into a specific kind of thing. Hawkins' reference frame organizes sensory data into a specific kind of object. Both are informational structures prior to any particular physical instantiation. This deepens the form-as-information bridge beyond the Levin connection already noted in FINDING-005. Flag for Stump Agent.

**C2A2 Architecture (META — highest priority):** The 13-agent system IS a Thousand Brains system. See PRS-CANDIDATE-01 and PRS-CANDIDATE-04. These are actionable design recommendations, not just theoretical observations.

**Friston (deepens existing flag):** The CMP (Cortical Messaging Protocol) may be a concrete implementation of message passing in Friston's variational inference framework. Each dispatch between modules is a variational message — an approximate posterior passed between Markov blankets. This would make the CMP = variational message passing, deepening the Friston × Hawkins connection beyond "both involve prediction loops."

**Wolfram (deepens existing flag):** Hawkins' reference frames as the medium for all knowledge echo Wolfram's ruliad as the space of all computation. Each cortical column explores a region of reference-frame space; each observer in the ruliad explores a region of computational space. The Thousand Brains system is a biological implementation of observer-dependent exploration of the ruliad.
